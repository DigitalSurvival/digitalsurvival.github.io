Stage7


What is Stage7?

Stage 7 is (or will be) the perfect installer for Gentoo Linux. It is build of the foundations of Andrew Gaffney's Quickstart project.



When can I have access to Stage7?

The software is in development, but beauty of GitHub is that you can have access to it right now! Simply fork or clone the Stage7 repository. If you are only interested in a finished version, you should dismay. An actual due date is not set, as I am only able to work on the project in my spare time (I work a full time job and I am actively involved at my church).


How can I help?

There are lots of ways you can help with the project. Having this page (and the Digital Survival) website re-written beautifully would be a big way you can help... You should e-mail me if you have any questions, concerns, feature requests, or suggestions! You can do so by combining my GitHub user name and @gmail.com together.
